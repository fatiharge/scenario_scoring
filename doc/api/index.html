Scenario Scoring

A dynamic software that scores certain types of data and generates simple reports.

Description

This project is about creating a dynamic software that scores certain types of data and generates simple reports. The software will be able to score data based on a set of rules and generate a report based on the scores. The software will be able to handle different types of data and rules, and generate reports in different formats.

Dependencies

  • Dart: '>=3.3.3 <4.0.0'
  • Flutter SDK
  • very_good_analysis: ^5.1.0
