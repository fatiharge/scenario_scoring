ArrayFilterExtension extension

ArrayFilterExtension extension

This extension is used to add additional functionality to the ArrayFilter class. It adds three methods:

  • summingFilter: This method filters the array Array using the objectFilter's filter method, multiplies the success score by the length of the filtered array, and returns whether the filtered array is not empty.
  • containsFilter: This method filters the array Array using the objectFilter's filter method, and returns whether the filtered array is not empty.
  • getFilteredArrayWithWhere: This method filters the array Array using the objectFilter's filter method and returns the filtered array.
