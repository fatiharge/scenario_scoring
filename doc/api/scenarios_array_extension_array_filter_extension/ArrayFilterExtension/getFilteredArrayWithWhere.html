getFilteredArrayWithWhere method

This method applies the filter defined in the objectFilter to each element in the array t. It then creates a new array that only includes the elements that meet the filter's conditions. This new array is then returned.

Implementation

Array getFilteredArrayWithWhere(Array t) {
  return t.where(
    (element) {
      return objectFilter.filter(t: element);
    },
  ).toList();
}
